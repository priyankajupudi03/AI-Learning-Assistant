# AI-Driven Emotion Detection & Personalized Learning Support Platform

## Project Overview

This project helps learners receive emotion-aware academic support. It uses Google Gemini AI to analyze student study-related questions and generate supportive, personalized responses. User interactions are stored in a CSV file for tracking and future analysis.

## Features

- Streamlit-based user interface
- Google Gemini AI integration
- Personalized learning guidance
- CSV interaction logging
- Environment setup using Python virtual environment
- Modular project structure

## Technologies Used

- Python
- Streamlit
- Google Gemini AI
- PyTorch
- Transformers
- Pandas
- NumPy
- python-dotenv
